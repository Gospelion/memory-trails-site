# Memory Trails Site

Standalone website for the Memory Trails ecosystem.

The initial site content was migrated from the `Where-Was-I` monorepo website at `apps/site`.

## Commands

- `npm run dev` starts the local Next.js development server.
- `npm run build` builds the production site.
- `npm run start` serves a production build.

## Product Structure

- Site pages live in `src/app`.
- Shared ecosystem and product metadata lives in `src/lib/products.ts`.
- Static brand and product assets live in `public`.
